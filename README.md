# 🎧 AI Mood-Based Playlist Generator

An interactive web application that recommends songs based on the user's mood, preferred language, and song type. The app provides a simple and elegant interface where users can select their mood and instantly get a song recommendation with a direct link to listen on Spotify.

---

## 🚀 Features

- 🎭 Select mood (Happy, Sad, Travel, etc.)
- 🌐 Choose language (English, Hindi, Marathi, Punjabi, Korean)
- 🎶 Select song type (Pop, Romantic, Travel, etc.)
- 🎧 Get a song recommendation instantly
- 🔗 Open the song directly on Spotify
- 🎨 Clean and elegant user interface

---

## 🛠 Tech Stack

- Python
- Streamlit
- Spotify Search Integration
- Custom CSS for UI Styling

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/ai-mood-playlist-generator.git
```

2. Navigate to the project folder

```bash
cd ai-mood-playlist-generator
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

Run the Streamlit app using:

```bash
streamlit run app.py
```

The application will open in your browser at:

```
http://localhost:8501
```

## 💡 How It Works

1. The user selects a mood.
2. The user chooses a preferred language.
3. The user selects the type of song.
4. The system randomly recommends a song from the dataset.
5. A Spotify search link is generated to listen to the recommended track.
